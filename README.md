## ✏️ Code convention

- feat: 새로운 기능 추가
- fix: 버그 수정
- docs: 문서 수정 (코드 변경 없음)
- style: 코드 스타일 수정 (포맷팅, 세미콜론 등)
- refactor: 리팩토링 (기능 변화 없음)
- chore: 빌드 설정, 패키지 매니저 설정 등 기타 잡일
- perf: 성능 개선
